- docker 
```
docker run -it -p 3000:3000 articodetz/engage-gpt:latest
```

- node 
``` 
git clone  https://github.com/tianzeshi-study/engage-gpt.git
cd  engage-gpt
yarn install 
yarn  start 
```
- dockerfile 
```
docker build . -t  <tag name>
```
